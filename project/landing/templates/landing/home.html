ProyectoIoT2025_10MJ

Sistema de gestión de sensores ambientales con operaciones CRUD integradas

Descripción del Proyecto

Este sistema permite la gestión de sensores, conectándose a una base de datos externa mediante un broker de seguridad.

Al iniciar sesión, los usuarios pueden consultar, crear, actualizar o eliminar sensores registrados en la base de datos conectada a través del Orion Context Broker.

Características principales del sistema:

  • Conexión segura con la base de datos de sensores a través de un broker de seguridad
  • Operaciones CRUD sobre sensores disponibles solo para usuarios autenticados
  • Interfaz web sencilla para visualizar y administrar sensores ambientales
